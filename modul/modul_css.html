Modul CSS: Referensi Sintaks Dasar

Pelajari cara kerja Selektor, Properti, dan Konsep Box Model di CSS.

← Kembali ke Halaman Utama

Penjelasan:

Sintaks CSS adalah sebuah aturan yang terdiri dari **Selector** (elemen yang dipilih), diikuti oleh **Declaration Block** yang berisi pasangan Properti dan Nilai.

Penggunaan:
Output:

Semua elemen paragraf (p) akan memiliki teks berwarna merah dan ukuran font 16px:

Note Penting:

Setiap deklarasi (pasangan Properti: Nilai) harus diakhiri dengan titik koma (;).

Penjelasan:

Selektor tipe menggunakan nama tag HTML secara langsung, misalnya h1 atau p. Selektor ini akan memengaruhi **semua** elemen dengan nama tag tersebut di halaman.

Penggunaan:
Output:

Semua elemen <p> di halaman akan berubah warna menjadi ungu.

Note Penting:

Selektor ini berguna untuk membuat gaya dasar yang konsisten di seluruh halaman.

Penjelasan:

Selektor kelas diawali dengan titik (.). Ini adalah cara yang paling fleksibel untuk menerapkan gaya karena dapat digunakan berulang kali pada elemen mana pun.

Penggunaan:
Output:

Hanya elemen yang memiliki atribut class="highlight" yang akan memiliki latar belakang kuning:

Note Penting:

Satu elemen HTML dapat memiliki **banyak** kelas (contoh: class="btn btn-primary"), dipisahkan oleh spasi.

Penjelasan:

Selektor ID diawali dengan tanda pagar (#). Selektor ini digunakan untuk menargetkan **satu elemen unik** di seluruh halaman.

Penggunaan:
Output:

Hanya satu elemen (biasanya Header Utama) yang akan terpengaruh dengan gaya ini:

Note Penting:

Karena ID harus unik, selektor ini memiliki prioritas (**Spesifisitas**) lebih tinggi daripada Selektor Kelas.

Penjelasan:

Setiap elemen HTML dianggap sebagai kotak (Box). Kotak ini terdiri dari empat lapisan yang bertumpuk: **Content**, **Padding** (ruang dalam), **Border** (garis tepi), dan **Margin** (ruang luar).

Penggunaan:
Content + Padding + Border
Output:

Ilustrasi visual lapisan Box Model:

Note Penting:

Secara *default*, properti width dan height hanya memengaruhi **Content**. Untuk mengubahnya, gunakan box-sizing: border-box;.

Penjelasan:

Properti color digunakan untuk mengatur warna teks dari suatu elemen.

Penggunaan:
Output:

Teks di dalam elemen yang ditargetkan akan berubah warna menjadi yang ditentukan:

Note Penting:

Warna dapat didefinisikan menggunakan nama warna (misalnya red), kode heksadesimal (misalnya #FF0000), atau nilai RGB/RGBA.

Penjelasan:

Properti background-color digunakan untuk mengatur warna latar belakang (background) dari suatu elemen.

Penggunaan:
Output:

Elemen akan diisi dengan warna latar belakang yang ditentukan:

Note Penting:

Properti ini tidak memengaruhi warna teks. Untuk warna teks, gunakan properti color.

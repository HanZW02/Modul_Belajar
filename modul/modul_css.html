Modul CSS: Referensi Sintaks Lengkap

Panduan lengkap tentang Selektor, Model Kotak (Box Model), dan Properti Dasar CSS.

Bagian 1: Struktur Dasar dan Penerapan

Penjelasan:

Aturan penulisan gaya di CSS punya tiga bagian utama: Selector (penunjuk elemen), Properti (apa yang mau diubah, misalnya warna), dan Nilai (gaya barunya).

Penggunaan:

p {
  color: red; /* Properti: Nilai */
  font-size: 16px;
}
Output:

Semua paragraf (p) akan memiliki teks berwarna merah dan ukuran font 16px.

Note Penting:

Setiap aturan gaya harus diakhiri dengan titik koma (;).

Penjelasan:

Ada tiga cara untuk menghubungkan CSS ke HTML. Cara terbaik adalah **External** (memisahkan kode) agar kode Anda lebih rapi dan mudah diatur.

Penggunaan:

<!-- 1. External (Paling disarankan) -->
<link rel="stylesheet" href="styles.css">

<!-- 2. Internal (di dalam tag style) -->
<style> p { color: blue; } </style>

<!-- 3. Inline (langsung di tag HTML) -->
<p style="color: green;">Teks</p>
Output:

Gaya yang paling spesifik akan menang. Contoh: Inline akan mengalahkan Internal.

Note Penting:

Prioritas terkuat ada pada **Inline CSS**. Sebisa mungkin hindari Inline CSS.

Penjelasan:

Satuan menentukan ukuran. Ada yang **Absolute** (nilainya tetap, seperti px) dan **Relative** (nilainya berubah tergantung elemen lain, seperti rem atau em).

Penggunaan:

.element-px { width: 300px; } /* 300 Pixel */
.element-rem { font-size: 1.2rem; } /* 1.2 kali ukuran font dasar */
.element-percent { width: 50%; } /* 50% dari lebar wadah (parent) */
Note Penting:

Menggunakan satuan **Relative** (rem) sangat disarankan agar desain Anda otomatis menyesuaikan di perangkat seluler (responsif).

Bagian 2: Selektor (Pemilih)

Penjelasan:

Selektor tipe menggunakan **nama tag HTML** langsung (misalnya p). Selektor ini akan mengatur gaya **semua** elemen dengan nama tag tersebut.

Penggunaan:

p {
  color: purple;
}
Output:

Semua paragraf (<p>) di halaman akan berubah warna menjadi ungu.

Note Penting:

Selektor ini adalah yang paling tidak spesifik. Artinya, gaya ini sangat mudah ditimpa oleh gaya lain (seperti Class atau ID).

Penjelasan:

Selektor kelas diawali dengan titik (.). Kelas ini bisa digunakan berulang kali pada elemen HTML mana pun. Ini yang paling sering digunakan untuk membuat desain.

Penggunaan:

/* Di CSS */
.highlight {
  background-color: yellow;
}

/* Di HTML */
<p class="highlight">Teks</p>
Output:

Hanya elemen yang menggunakan class="highlight" yang akan memiliki latar belakang kuning.

Note Penting:

Satu elemen HTML bisa memiliki **banyak** kelas (contoh: class="btn btn-primary"), dipisahkan oleh spasi.

Penjelasan:

Selektor ID diawali dengan tanda pagar (#). ID hanya boleh digunakan untuk menargetkan **satu elemen unik** saja di seluruh halaman.

Penggunaan:

/* Di CSS */
#main-header {
  text-align: center;
}

/* Di HTML */
<h1 id="main-header">Judul Utama</h1>
Output:

Hanya satu elemen yang memiliki ID tersebut yang akan terpengaruh.

Note Penting:

ID memiliki prioritas gaya tertinggi dan tidak boleh diduplikasi.

Penjelasan:

Beberapa selektor khusus untuk menargetkan elemen secara spesifik:

  • **Universal:** * Memilih semua elemen di halaman.
  • **Attribut:** [type="text"] Memilih elemen berdasarkan atributnya (misalnya semua input teks).
  • **Grouping:** h1, h2, p Memilih banyak elemen sekaligus untuk diberi gaya yang sama (dipisahkan koma).
Penggunaan:

/* Memilih semua elemen */
* { margin: 0; padding: 0; } 

/* Mengatur gaya semua input tipe submit */
input[type="submit"] {
  background-color: blue;
}
Note Penting:

Grouping selector membantu mengurangi jumlah baris kode CSS karena Anda tidak perlu mengulang gaya yang sama untuk elemen berbeda.

Bagian 3: Model Kotak (Box Model) dan Tata Letak

Penjelasan:

Setiap elemen HTML dianggap sebagai kotak. Kotak ini punya 4 lapisan dari dalam ke luar: Content, Padding (ruang dalam), Border (garis tepi), dan Margin (ruang luar).

Penggunaan:

.box {
  width: 200px;
  padding: 20px; /* Ruang antara konten dan border */
  border: 5px solid red;
  margin: 10px; /* Ruang di luar border */
}
Content + Padding + Border
Output:

Ilustrasi visual lapisan Box Model.

Note Penting:

Gunakan box-sizing: border-box; untuk mempermudah perhitungan, karena total lebar elemen akan tetap sesuai nilai width yang Anda tentukan.

Penjelasan:

Properti margin mengatur ruang di **luar** elemen (lapisan terluar Box Model). Fungsinya untuk memisahkan satu elemen dari elemen lainnya.

Penggunaan:

/* Aturan Singkat (Top | Right | Bottom | Left) */
margin: 10px 20px 30px 40px; 

/* Untuk Rata Tengah Horizontal */
margin: 0 auto;
Output:

Spasi transparan akan ditambahkan di luar batas elemen.

Note Penting:

Untuk membuat elemen blok berada di tengah halaman, gunakan width dan margin: auto;.

Penjelasan:

Properti padding mengatur ruang di **dalam** elemen (antara konten dan border). Fungsinya untuk memberi bantalan agar konten tidak terlalu menempel pada tepi.

Penggunaan:

/* Aturan Singkat (Top | Right | Bottom | Left) */
padding: 15px 15px 15px 15px;
Output:

Jarak antara konten dan batas (border) elemen akan diperlebar.

Note Penting:

Padding selalu mewarisi warna latar belakang (background color) elemen tersebut.

Penjelasan:

Properti border mengatur tampilan garis tepi di sekitar elemen. Anda bisa mengatur lebar, gaya, dan warna garis.

Penggunaan:

/* Singkatan: Lebar | Gaya | Warna */
border: 2px solid #0d6efd; 

/* Membuat sudut melengkung */
border-radius: 5px;
Output:

Elemen akan memiliki garis biru solid dengan sudut membulat.

Note Penting:

Anda harus selalu menentukan gaya (misal: solid) agar border terlihat. Pilihan gaya lain termasuk dashed, dotted, atau double.

Penjelasan:

Properti display mengatur bagaimana elemen diletakkan di halaman.

  • **block**: Mengisi seluruh lebar yang tersedia (seperti <p>, <div>). Selalu mulai di baris baru.
  • **inline**: Hanya selebar konten (seperti <span>, <a>). Berada di baris yang sama.
  • **none**: Menyembunyikan elemen dan ruang yang ditempatinya.
Penggunaan:

.inline-item { display: inline; }
.hidden { display: none; }
Output:

Ilustrasi bagaimana elemen blok dan inline ditampilkan di halaman.

Note Penting:

Elemen inline tidak bisa diatur width dan height-nya.

Penjelasan:

Properti position mengatur posisi elemen. Ini bekerja bersama properti top, bottom, left, dan right.

Penggunaan:

.parent { position: relative; }

.child {
  position: absolute;
  top: 10px; /* 10px dari tepi atas parent */
}
Note Penting:

Posisi absolute dipengaruhi oleh elemen **parent terdekat** yang memiliki position: relative;.

Bagian 4: Warna dan Font

Penjelasan:

Properti color mengatur warna teks. Properti background-color mengatur warna latar belakang elemen.

Penggunaan:

h1 {
  color: #0d6efd; /* Warna teks (Kode Hex) */
  background-color: rgb(240, 240, 240); /* Warna latar */
}
Output:

Teks berwarna biru dengan latar belakang abu-abu muda.

Note Penting:

Warna bisa diisi dengan **Nama** (misal: red), **Hex** (misal: #FF0000), atau **RGB/RGBA**.

Penjelasan:

Properti font-family menentukan jenis huruf (font) yang akan digunakan.

Penggunaan:

body {
  font-family: Arial, Helvetica, sans-serif; /* Fallback jika Arial tidak ada */
}
Output:

Teks akan ditampilkan menggunakan jenis font "Arial" atau fallback "sans-serif".

Note Penting:

Selalu sediakan font cadangan (*fallback font*) agar tampilan di setiap browser tetap konsisten.

Penjelasan:
  • font-size: Mengatur ukuran teks (gunakan px, em, atau rem).
  • font-weight: Mengatur ketebalan teks (misal: bold, normal, atau nilai angka 100-900).
Penggunaan:

.judul {
  font-size: 2.5rem;
  font-weight: 700; /* 700 = Tebal */
}
Note Penting:

Nilai angka 400 setara dengan normal dan 700 setara dengan bold.

Penjelasan:
  • text-align: Mengatur perataan teks (left, right, center, justify).
  • text-decoration: Menambahkan atau menghapus garis dekorasi pada teks (misal: underline atau none).
Penggunaan:

.center-text {
  text-align: center;
}

a {
  text-decoration: none; /* Menghapus garis bawah pada tautan */
}
Output:

Teks akan berada di tengah kontainer, dan tautan tidak memiliki garis bawah.

Note Penting:

text-decoration: none; sangat umum digunakan pada tautan (<a>) untuk menghapus gaya bawaan browser.

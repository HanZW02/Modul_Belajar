Modul CSS: Dasar-Dasar Penataan Gaya

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan gaya elemen-elemen HTML.

Sintaks Dasar:

CSS terdiri dari **selektor** (misalnya `p`), **properti** (misalnya `color`), dan **nilai** (misalnya `red`). Setiap pasangan properti dan nilai adalah sebuah Deklarasi, dan Deklarasi berada di dalam kurung kurawal `{}`.

Visualisasi Sintaks:

Output Sintaks:

Semua teks dalam elemen <p> akan berwarna merah dan memiliki ukuran font 16 piksel.

Ada tiga cara utama untuk menerapkan CSS ke dokumen HTML:

  1. **Inline CSS:** Menggunakan atribut style langsung pada elemen HTML.
  2. **Internal CSS:** Menggunakan tag <style> di dalam <head>.
  3. **External CSS:** Menghubungkan file .css eksternal (Metode terbaik, digunakan oleh Bootstrap).

Output Sintaks:

Output visualnya menunjukkan perbedaan prioritas (cascading) gaya antara ketiga metode tersebut.

Selain selector nama elemen, CSS sering menggunakan **Class** (diawali dengan titik .): untuk menata banyak elemen, dan **ID** (diawali dengan pagar #): untuk menata satu elemen unik.

Visualisasi Sintaks:

Output Sintaks:

Elemen dengan class akan memiliki latar belakang biru. Elemen dengan ID akan memiliki teks tebal.

Setiap elemen HTML dianggap sebagai kotak. Kotak ini terdiri dari empat bagian yang bertumpuk: **Content**, **Padding**, **Border**, dan **Margin**.

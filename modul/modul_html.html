Modul HTML: Dasar-Dasar Struktur Web

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

← Kembali ke Halaman Utama

Sintaks Inti:

Sintaks inti HTML mencakup deklarasi <!DOCTYPE html>, elemen <html>, <head> (untuk metadata), dan <body> (untuk konten yang terlihat).

Visualisasi Sintaks:

Output Sintaks:

Outputnya adalah halaman web kosong dengan judul yang muncul di tab browser.

Sintaks Heading (H1-H6) dan Paragraf (P):

Heading (<h1> hingga <h6>) digunakan untuk judul dengan ukuran yang berbeda. Paragraf (<p>) digunakan untuk teks biasa. Teks juga bisa dibuat **tebal** (<strong>) atau *miring* (<em>).

Visualisasi Sintaks:

Output Sintaks:

Heading (H1) ditampilkan paling besar, H3 lebih kecil. Teks dalam **strong** akan tebal, dan *em* akan miring.

Sintaks Tautan (<a>) dan Gambar (<img>):

Tautan menggunakan <a> dengan atribut href untuk menentukan tujuan. Gambar menggunakan elemen <img> dengan atribut src untuk sumber file dan alt untuk teks alternatif.

Visualisasi Sintaks:

Output Sintaks:

Akan ada teks tautan yang dapat diklik (membuka di tab baru) dan di bawahnya akan muncul gambar logo ekstrakurikuler dengan lebar 100px.

Sintaks Daftar:

Daftar Tak Berurutan (<ul>) menggunakan simbol (bullet point). Daftar Berurutan (<ol>) menggunakan angka atau huruf. Keduanya menggunakan elemen <li> untuk setiap itemnya.

Visualisasi Sintaks:

Output Sintaks:

Daftar tak berurutan akan menggunakan simbol, sedangkan daftar berurutan akan menggunakan angka.

Bagian 1: Materi Dasar Struktur & Konten

Penjelasan:

Deklarasi <!DOCTYPE html> harus menjadi baris pertama dalam kode HTML kamu. Tujuannya adalah memberi tahu browser bahwa ini adalah dokumen HTML5. Deklarasi ini merupakan sebuah instruksi kunci yang membuat browser masuk ke Standards Mode (Mode Standar), memastikan kode HTML dan CSS dibaca dan ditampilkan secara benar sesuai dengan standar World Wide Web Consortium (W3C), sehingga tampilan website menjadi konsisten di berbagai browser modern. Jika deklarasi ini dihilangkan, browser bisa saja masuk ke Quirks Mode (Mode Aneh) yang akan menyebabkan tampilan halaman menjadi tidak terduga atau rusak karena meniru perilaku browser-browser kuno.

Penggunaan:

<!DOCTYPE html>
<html>
<head>
  ...
</head>
<body>
  ...               
</body>
</html>
Note Penting:

Ini bukan tag HTML, melainkan instruksi untuk browser dan harus diletakkan di baris paling awal. Jadi, tidak perlu untuk membuat </!DOCTYPE html>

Penjelasan:

<html> adalah elemen akar (root element) yang wajib ada dan membungkus seluruh konten dari sebuah halaman HTML, di luar deklarasi <!DOCTYPE html>. Elemen ini berfungsi sebagai wadah utama yang menandai awal dan akhir dari dokumen HTML secara keseluruhan, memberitahu web browser bahwa semua yang ada di dalamnya adalah kode HTML. Tag ini juga berfungsi sebagai tempat untuk menetapkan atribut global, yang paling umum adalah atribut lang (contoh: <html lang="id">). Atribut ini sangat penting karena menentukan bahasa utama konten halaman (dalam contoh ini, Indonesia), yang membantu mesin pencari (SEO) untuk mengindeks halaman dengan benar dan membantu perangkat pembaca layar (teknologi aksesibilitas) untuk membacakan teks dengan intonasi yang tepat. Semua elemen penting lainnya, yaitu <head> dan <body>, harus diletakkan sebagai anak langsung dari tag <html> dan </html>.

Penggunaan:

<!DOCTYPE html> 
<html lang="id">  
<head>
        
</head>
<body>
        
</body>
</html>
Note Penting:

Sertakan atribut lang (contoh: lang="id") untuk menentukan bahasa dokumen.

Penjelasan:

Tag <head> berfungsi sebagai wadah metadata atau informasi tentang dokumen HTML itu sendiri. Letaknya harus berada tepat di dalam tag <html>, sebelum tag <body>. Metadata adalah data tentang data; dalam konteks web, ini adalah informasi yang tidak ditampilkan secara langsung pada halaman web oleh browser kepada pengguna, tetapi sangat penting untuk berbagai fungsi.

Penggunaan:

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
Output:

pada baris kode ini hanya teks dalam <title> yang terlihat di tab browser.

Note Penting:

Elemen-elemen penting di dalamnya termasuk <title>, <meta>, dan <link> ke CSS.

Penjelasan:

Tag <body> adalah tag yang paling penting dalam sebuah dokumen HTML karena berfungsi sebagai wadah utama untuk semua konten yang terlihat oleh pengguna web. Tag ini harus diletakkan setelah tag <head> dan merupakan anak langsung dari tag <html>.

Penggunaan:
                  
<!DOCTYPE html>
<html>
<body>
  <h1>Judul ini akan terlihat di body</h1>
  <p>Paragraf ini juga akan terlihat di body.</p>
</body>
</html>
Output:

Semua elemen seperti heading dan paragraf akan terlihat di halaman:

Note Penting:

Setiap dokumen HTML hanya boleh memiliki satu tag <body>.

Penjelasan:

<p> adalah salah satu elemen yang ada pada HTML yang digunakan untuk mendefinisikan sebuah paragraf dalam dokumen web. Tag ini merupakan salah satu elemen blok level yang paling sering digunakan dalam HTML. Secara default, tag ini memberikan spasi vertical sebelum dan sesudah teks, sehingga memisahkan paragraf satu dengan lainnya. Dengan menggunakannya maka akan menghasilkan teks yang lebih rapi dan mudah dibaca oleh pengunjung website.

Penggunaan:
                  
<!DOCTYPE html>
<html>
<body>
  <p>ini adalah teks!</p>
  <p>ini juga teks!</p>
</body>
</html>
Output:

Dua blok teks terpisah dengan jarak vertikal di antaranya:

Note Penting:

Jangan gunakan <p> untuk menyusun list atau elemen non-paragraf lainnya.

Penjelasan:

Tag <a> (anchor) membuat hyperlink. Elemen ini mengubah teks atau objek menjadi tautan yang dapat diklik.

Penggunaan:

Ketika kalian klik link dari <a></a>, kalian bakal dialihkan ke link yang kalian ketik di href.

                  
<!DOCTYPE html>
<html>
<body>
  <a href="https://www.instagram.com/rhnkmlw_/" target="_blank">Follow IG Pak Rehan yes!!!</a>
</body>
</html>
Output:

Teks yang diklik akan mengarahkan ke URL tujuan:

Note Penting:

Atribut href adalah wajib untuk menentukan alamat tujuan. Gunakan target="_blank" untuk membuka tautan di tab baru.

Penjelasan:

Tag <img> menyematkan gambar ke dalam halaman.

Penggunaan:

lorem ipsum

                  
<!DOCTYPE html>
<html>
<body>
  <img src="*ganti dengan image mu*" alt="*Deskripsi Gambar*">
</body>
</html>
Output:

Gambar yang disematkan akan muncul di halaman:

Note Penting:

Atribut src dan alt adalah wajib. Gunakan width dan height untuk menentukan ukuran.

Penjelasan:

Tag <hr> merepresentasikan garis pemisah tematik (misalnya pergantian subjek) dan ditampilkan sebagai garis horizontal. Ini adalah tag *self-closing*.

Penggunaan:
Output:

Garis horizontal penuh akan muncul di antara konten:

Note Penting:

Ini adalah tag *self-closing* dan digunakan untuk pemisahan konten, bukan untuk dekorasi.

Penjelasan:

Tag <br> menyisipkan jeda baris tunggal (line break). Berguna untuk puisi atau alamat di mana pemisahan baris penting. Ini adalah tag *self-closing*.

Penggunaan:
Output:

Teks akan dilanjutkan di baris baru tanpa membuat paragraf baru:

Note Penting:

Jangan gunakan beberapa tag <br> berturut-turut untuk menambah spasi vertikal. Gunakan CSS Margin atau Padding pada elemen lain.

Penjelasan:

Tag <pre> menampilkan teks dengan format dan spasi yang dipertahankan persis seperti saat diketik dalam kode sumber. Menggunakan font monospace.

Penggunaan:
Output:

Teks akan ditampilkan persis seperti format yang Anda ketik, termasuk spasi dan baris baru:

Note Penting:

Sangat berguna untuk menampilkan blok kode pemrograman atau ASCII Art.

Bagian 2: Format Teks dan Atribut

Penjelasan:

HTML menyediakan tag khusus untuk memformat teks:

  • **Visual (Tidak Semantik):** <b> (Bold), <i> (Italic).
  • **Semantik (Penting):** <strong> (Teks penting), <em> (Penekanan).
  • **Lainnya:** <mark> (Highlight), <del> (Teks dicoret), <ins> (Teks baru), <sub> (Subscript), <sup> (Superscript).
Penggunaan:
Output:

Hasil visual dari semua tag pemformatan teks:

Note Penting:

Sebaiknya selalu gunakan tag **semantik** (<strong> dan <em>) daripada tag visual (<b> dan <i>) karena lebih baik untuk SEO dan aksesibilitas.

Penjelasan:

Komentar digunakan untuk menjelaskan kode, meninggalkan catatan, atau menonaktifkan kode sementara. Komentar **sepenuhnya diabaikan** oleh browser.

Penggunaan:
Output:

Komentar tidak muncul di halaman, hanya terlihat saat melihat sumber halaman (*view page source*).

Note Penting:

Selalu gunakan komentar untuk membuat kode Anda lebih mudah dipahami oleh orang lain (atau diri Anda sendiri di masa depan).

Penjelasan:

Atribut Global dapat digunakan pada hampir semua elemen HTML untuk memberikan informasi tambahan atau memodifikasi tampilan/perilaku elemen.

Penggunaan:
Note Penting:
  • class: Untuk menata gaya sekelompok elemen (utama untuk CSS/Bootstrap).
  • id: Untuk mengidentifikasi satu elemen unik (penting untuk JavaScript).
  • style: Untuk styling CSS secara inline.
  • title: Menampilkan teks tooltip saat kursor diarahkan ke elemen.

Penjelasan:

Beberapa tag memiliki atribut yang wajib atau sangat penting untuk fungsionalitasnya.

Penggunaan:
Note Penting:
  • **<a>**: target="_blank" membuka tautan di tab baru.
  • **<img>**: alt harus menjelaskan gambar dan sangat penting untuk aksesibilitas.
  • Atribut width dan height pada <img> membantu browser merender halaman lebih cepat.

Penjelasan:

Tag <style> digunakan untuk menulis aturan CSS langsung di dalam dokumen HTML. Tag ini ditempatkan di dalam bagian <head>.

Penggunaan:
Output:

Aturan CSS di dalamnya akan mengubah tampilan elemen HTML.

Note Penting:

Penggunaan tag <style> hanya disarankan untuk sedikit aturan. Untuk styling yang lebih besar dan terorganisir, gunakan file eksternal.

Penting: Sintaks dan cara penulisan kode CSS secara lengkap dapat Anda pelajari di **Modul CSS**.

Bagian 3: Struktur Konten, Daftar, & Tabel

Penjelasan:

Tag heading digunakan untuk menentukan pentingnya judul pada hierarki konten. <h1> adalah yang paling penting (terbesar), dan <h6> paling rendah.

Penggunaan:
Output:

Ukuran teks yang berbeda untuk setiap tingkatan:

Note Penting:

Idealnya, setiap halaman hanya memiliki **satu** tag <h1>.

Penjelasan:

Tag <div> adalah elemen kontainer tingkat blok (block-level) yang paling umum. Digunakan untuk mengelompokkan elemen lain untuk tujuan *styling* (dengan CSS).

Penggunaan:
Output:

Konten akan dikelompokkan dan dapat diterapkan gaya sebagai satu kesatuan blok.

Note Penting:

Tag <div> adalah tag non-semantik. Gunakan tag semantik HTML5 (seperti <section> atau <article>) jika memungkinkan.

Penjelasan:

Tag <span> adalah kontainer generik tingkat *inline*. Digunakan untuk mengelompokkan sebagian kecil teks atau elemen *inline* lainnya untuk *styling* tanpa memulai baris baru.

Penggunaan:
Output:

Hanya teks yang berada dalam <span> yang terpengaruh tanpa mengganggu aliran teks di sekitarnya.

Note Penting:

Tidak seperti <div>, <span> tidak dapat menampung elemen blok level lain secara langsung.

Penjelasan:

Tag semantik (seperti <header>, <nav>, <footer>, <section>, <article>) memberikan makna pada bagian-bagian halaman, membantu mesin pencari dan alat bantu aksesibilitas memahami struktur konten.

Penggunaan:
Output:

Secara visual tampak seperti <div>, tetapi secara struktural jauh lebih baik untuk SEO.

Note Penting:

Penggunaan tag semantik harus diprioritaskan di atas <div> untuk tata letak utama.

Penjelasan:

<ul> (Unordered List) digunakan untuk membuat daftar yang tidak mementingkan urutan. Setiap item dalam daftar diapit oleh tag <li> (List Item).

Penggunaan:
Output:

Daftar akan ditampilkan dengan simbol (default: bullet point) di depan setiap item.

Note Penting:

Untuk mengubah simbol *bullet*, gunakan properti CSS list-style-type.

Penjelasan:

<ol> (Ordered List) digunakan untuk membuat daftar yang urutannya penting. Setiap item juga diapit oleh tag <li>.

Penggunaan:
Output:

Daftar akan ditampilkan dengan penomoran otomatis (default: angka) di depan setiap item.

Note Penting:

Anda dapat mengubah jenis penomoran menggunakan atribut type (misalnya type="A" untuk huruf kapital).

Penjelasan:

Tag <table> mendefinisikan tabel data. Tag ini berfungsi sebagai wadah untuk semua elemen tabel lainnya.

Penggunaan:
Output:

Tabel yang belum diberi gaya CSS (tampak polos).

Note Penting:

Selalu gunakan tag semantik <thead>, <tbody>, dan <tfoot>.

Penjelasan:

Tag-tag ini menyusun isi tabel: <tr> (Table Row), <td> (Table Data), dan <th> (Table Header).

Penggunaan:
Output:

Sel header (th) secara otomatis akan dicetak tebal.

Note Penting:

Gunakan atribut colspan atau rowspan pada <td> atau <th> untuk menggabungkan sel.

Bagian 4: Formulir Interaktif

Penjelasan:

Tag <form> adalah wadah untuk semua elemen input pengguna. Tag ini mengumpulkan data pengguna dan mengirimkannya ke server.

Penggunaan:
Output:

Kontainer yang menampung semua elemen input formulir.

Note Penting:
  • action: URL (alamat server) tempat data formulir akan dikirim.
  • method: Metode HTTP yang digunakan (umumnya GET atau POST).

Penjelasan:

<input> adalah elemen paling serbaguna untuk menerima data. <label> memberikan deskripsi pada input.

Penggunaan:
Output:

Kotak teks standar di halaman yang siap diisi.

Note Penting:

Hubungkan <label> ke <input> menggunakan atribut for dan id yang cocok.

Penjelasan:

<button> digunakan untuk tombol yang dapat diklik. <textarea> digunakan untuk menerima input teks multi-baris (misalnya komentar).

Penggunaan:
Output:

Tombol kirim (submit) dan kotak besar untuk komentar muncul di halaman.

Note Penting:

Selalu tentukan atribut type pada <button>.

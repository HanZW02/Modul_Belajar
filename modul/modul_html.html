Pelajari fungsi setiap elemen HTML, struktur dokumen, dan format teks.

Contoh Struktur Dasar Dokumen HTML5 Sederhana

          
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Halaman</title>
</head>
<body>
  <h1>Halo, Dunia!</h1>
  <p>Selamat datang di modul HTML.</p>
</body>
</html>

Bagian 1: Materi Dasar Struktur & Konten

Penjelasan:

Kode <!DOCTYPE html> wajib ditulis di baris paling atas. Ini berfungsi memberi tahu browser: "Halo, saya menggunakan HTML5 (versi terbaru)!". Tanpa kode ini, tampilan websitemu mungkin akan terlihat berantakan atau aneh di beberapa browser karena dianggap menggunakan versi kuno.

Penggunaan:

<!DOCTYPE html>
<html>
<head>
  <!-- Informasi meta -->
</head>
<body>
  <!-- Isi website -->
</body>
</html>
Note Penting:

Ini bukan tag HTML biasa, jadi tidak perlu penutup seperti </!DOCTYPE>. Cukup tulis satu kali di awal file.

Penjelasan:

Tag <html> adalah "bungkus utama" dari seluruh kodinganmu. Semua kode lain (kecuali DOCTYPE) harus berada di dalam tag ini. Kita juga bisa memberi tahu bahasa utama website di sini (contoh: Indonesia atau Inggris).

Penggunaan:

<!DOCTYPE html> 
<html lang="id"> 
  <!-- Semua kode lain ada di dalam sini -->
</html>
Note Penting:

Gunakan lang="id" untuk website Bahasa Indonesia atau lang="en" untuk Inggris. Ini membantu Google memahami isi websitemu.

Penjelasan:

Tag <head> berisi informasi rahasia atau pengaturan website yang **tidak ditampilkan** langsung kepada pengunjung di halaman utama. Isinya biasanya judul tab browser, pengaturan karakter, dan link ke file CSS (hiasan).

Penggunaan:

<head>
  <meta charset="UTF-8">
  <title>Judul di Tab Browser</title>
</head>
Output:

Hanya teks di dalam <title> yang akan muncul di bagian paling atas tab browser (seperti Chrome/Edge).

Note Penting:

Jangan menaruh konten seperti paragraf atau gambar di dalam <head>. Tempatnya bukan di sini!

Penjelasan:

Tag <body> adalah tempat untuk semua hal yang ingin kamu tampilkan kepada pengunjung. Teks, gambar, video, tombol, tabel, semuanya wajib ditulis di dalam tag ini.

Penggunaan:
                  
<!DOCTYPE html>
<html>
<body>
  <h1>Judul Besar</h1>
  <p>Ini adalah paragraf yang bisa dibaca pengunjung.</p>
</body>
</html>
Output:

Semua elemen di dalam body akan terlihat di layar:

Note Penting:

Satu file HTML hanya boleh punya satu tag <body>.

Penjelasan:

<p> digunakan untuk membuat **paragraf**. Browser secara otomatis akan memberikan jarak (spasi) kosong di atas dan di bawah paragraf, supaya teks tidak menumpuk dan lebih enak dibaca.

Penggunaan:
                  
<body>
  <p>Ini adalah paragraf pertama.</p>
  <p>Ini adalah paragraf kedua.</p>
</body>
Output:

Dua blok teks terpisah dengan jarak vertikal:

Note Penting:

Jangan gunakan <p> jika kamu hanya ingin membuat judul besar (gunakan Heading).

Penjelasan:

Tag <a> (Anchor) berfungsi membuat teks atau gambar menjadi tombol link yang bisa diklik untuk pindah ke halaman lain atau website lain.

Penggunaan:

Isi atribut href="..." dengan alamat tujuan.

                  

<a href="https://www.instagram.com/rhnkmlw_/" target="_blank">Follow IG Pak Rehan yes!!!</a>
Output:

Teks biru bergaris bawah yang bisa diklik:

Note Penting:

Tambahkan target="_blank" jika ingin link terbuka di tab baru (supaya website kita tidak tertutup).

Penjelasan:

Tag <img> digunakan untuk menampilkan gambar di halaman web. Tag ini unik karena tidak punya penutup (tidak ada </img>).

Penggunaan:
                  
<img src="*ganti dengan direktori image mu*" alt="*Deskripsi Gambar*">
Ini adalah contoh dari menghubungkan direktori gambar pada src 
                  
<img src="materi/gambar/mrbean.jpg" alt="Gambar MrBean">
Output:
Note Penting:

Wajib isi alt="..."! Teks ini akan muncul jika gambar gagal dimuat (misal internet lambat) dan membantu teman-teman tunanetra memahami isi gambar.

Penjelasan:

Tag <hr> digunakan untuk membuat garis lurus mendatar pembatas. Biasanya dipakai untuk memisahkan antar topik cerita. Tag ini juga tidak punya penutup.

Penggunaan:

<p>Paragraf pertama tentang topik A.</p>
<hr>
<p>Paragraf kedua tentang topik B.</p>
Output:

Garis horizontal penuh akan muncul di antara konten:

Note Penting:

Gunakan ini untuk memisahkan konten secara tema, bukan sekadar untuk hiasan garis (untuk hiasan lebih baik pakai CSS).

Penjelasan:

Tag <br> (Break) berfungsi seperti menekan tombol Enter di keyboard. Ini memaksa teks turun ke baris baru tanpa membuat jarak spasi paragraf yang lebar. Cocok untuk menulis alamat atau puisi.

Penggunaan:

<p>
  Jalan Mawar No. 12<br>
  Jakarta Selatan,<br>
  Indonesia
</p>
Output:

Teks akan dilanjutkan di baris baru tanpa membuat paragraf baru:

Note Penting:

Jangan gunakan <br> berkali-kali hanya untuk membuat jarak kosong yang luas. Gunakan CSS Margin/Padding untuk itu.

Penjelasan:

Tag <pre> menampilkan teks persis apa adanya sesuai yang kamu ketik di kode, termasuk spasi yang banyak dan enter. Biasanya font-nya berubah menjadi gaya mesin ketik (monospace).

Penggunaan:

<pre>
  Teks ini akan    mempertahankan
      setiap spasi
  dan baris baru.
</pre>
Output:

Teks akan ditampilkan persis seperti format yang Anda ketik, termasuk spasi dan baris baru:

Note Penting:

Sangat berguna jika kamu ingin menampilkan contoh kode program (codingan) di websitemu.

Bagian 2: Format Teks dan Atribut

Penjelasan:

HTML menyediakan tag khusus untuk memformat teks:

  • Visual (Tidak Semantik): <b> (Bold), <i> (Italic).
  • Semantik (Penting): <strong> (Teks penting), <em> (Penekanan).
  • Lainnya: <mark> (Highlight), <del> (Teks dicoret), <ins> (Teks baru), <sub> (Subscript), <sup> (Superscript).
Penggunaan:

<p>
  <b>Tebal</b> (Bold/Teks Tebal) dan <strong>Penting</strong>(Teks Penting)
</p>
<p>
  <i>Miring</i> (Italic/Teks Miring) dan <em>Ditekankan</em> (Teks Penekanan)
</p>
<p>
  <mark>HighLight</mark> (Teks Highlight)
</p>
<p>
  <del>teks dicoret</del> (Teks Dicoret)
</p>
<p>
  <ins>teks dicoret</ins> (Teks Baru)
</p>
<p>
  H<sub>2</sub>O (Teks Subscript)
</p>
<p>
  E = mc<sup>2</sup> (Teks Superscript)
</p>
Output:

Hasil visual dari semua tag pemformatan teks:

Note Penting:

Sebaiknya selalu gunakan tag semantik (<strong> dan <em>) daripada tag visual (<b> dan <i>) karena lebih baik untuk SEO dan aksesibilitas.

Penjelasan:

Komentar adalah catatan rahasia untuk programmer. Tulisan yang ada di dalam komentar tidak akan muncul di browser. Ini berguna untuk memberi tanda atau pengingat pada kodinganmu sendiri.

Penggunaan:

Gunakan ini sebagai "note" kode kamu sendiri

Pada code ini <!-- --> tidak akan tampak di output, yang tampak pada kode ini hanya <p></p> saja 


<!-- Ini adalah komentar, tidak akan tampil di browser -->
<p>Ini paragraf yang tampil.</p>
Output:

Komentar tidak muncul di halaman.

Note Penting:

Selalu gunakan komentar untuk membuat kode kamu lebih mudah dipahami oleh orang lain (atau diri kamu sendiri di masa depan).

Penjelasan:

Atribut Global bisa dipasang di hampir semua tag. Dua yang paling sering dipakai adalah:

  • class: Memberi nama "grup" pada elemen. Banyak elemen bisa punya class yang sama. (Contoh: class="kotak-merah").
  • id: Memberi nama "unik" (seperti KTP). Dalam satu halaman, tidak boleh ada dua elemen dengan ID yang sama. (Contoh: id="tombol-login").
Penggunaan:

<p class="teks-merah">Paragraf ini merah.</p>
<p class="teks-merah">Ini juga merah (satu grup).</p>

<h1 id="judul-utama">Judul Spesial</h1>
Note Penting:
  • class: Untuk menata gaya sekelompok elemen (utama untuk CSS/Bootstrap).
  • id: Untuk mengidentifikasi satu elemen unik (penting untuk JavaScript).

Penjelasan:

Beberapa tag tidak akan berguna tanpa atribut khususnya.

  • Link <a> wajib punya href (alamat tujuan).
  • Gambar <img> wajib punya src (sumber file gambar).
Penggunaan:

<!-- Link ke Google -->
<a href="https://google.com">Ke Google</a>

<!-- Menampilkan Gambar -->
<img src="gambar.jpg" alt="Deskripsi Gambar">
Output:
Note Penting:
  • <a>: target="_blank" membuka tautan di tab baru.
  • <img>: alt harus menjelaskan gambar dan sangat penting untuk aksesibilitas.
  • Atribut width dan height pada <img> membantu browser merender halaman lebih cepat.

Penjelasan:

Tag <style> adalah tempat kita menulis kode CSS (kode untuk menghias warna, ukuran, layout) langsung di dalam file HTML. Biasanya diletakkan di bagian <head>.

Penggunaan:

<head>
  <style>
    h1 {
      color: blue; /* Mengubah judul jadi biru */
      font-size: 40px;
    }
  </style>
</head>
Output:

Aturan CSS di dalamnya akan mengubah tampilan elemen HTML.

Note Penting:

Ini cara cepat untuk menghias. Tapi untuk proyek besar, kita biasanya memisahkan kode CSS di file tersendiri agar rapi.

Bagian 3: Struktur Konten, Daftar, & Tabel

Penjelasan:

HTML menyediakan 6 level judul. <h1> adalah Judul Utama (Paling Besar & Penting), sedangkan <h6> adalah sub-judul terkecil.

Penggunaan:

<h1>Heading 1 (Paling Besar)</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6 (Paling Kecil)</h6>
Output:

Ukuran teks yang berbeda untuk setiap tingkatan:

Note Penting:

Gunakan hanya satu <h1> di setiap halaman website sebagai judul utama konten tersebut.

Penjelasan:

Tag <div> ibarat sebuah "kardus" atau kotak kosong. Kita menggunakannya untuk mengelompokkan beberapa elemen (misalnya judul + paragraf + gambar) menjadi satu grup, supaya mudah dipindahkan atau diwarnai background-nya.

Penggunaan:

<div style="background-color: lightgray; padding: 10px;">
  <h2>Artikel 1</h2>
  <p>Ini adalah isi artikel yang dibungkus dalam div.</p>
</div>
Output:

Konten akan dikelompokkan dan dapat diterapkan gaya sebagai satu kesatuan blok.

Note Penting:

Div adalah elemen "Block", artinya dia akan selalu membuat baris baru (memenuhi lebar layar).

Penjelasan:

Tag <span> mirip dengan div, tapi versi kecilnya. Span digunakan untuk menandai potongan kecil teks di dalam kalimat tanpa membuat baris baru. Cocok untuk mewarnai satu kata tertentu.

Penggunaan:

<p>Saya suka warna <span style="color: red;">Merah</span> sekali.</p>
Output:

Hanya kata "Merah" yang berubah warna, kalimat tetap tersambung.

Note Penting:

Tidak seperti <div>, <span> tidak dapat menampung elemen blok level lain secara langsung.

Penjelasan:

Tag Semantik adalah tag yang namanya menjelaskan fungsinya. Daripada cuma pakai <div> untuk semua hal, lebih baik pakai tag ini agar Google paham struktur websitemu:

  • <header>: Untuk bagian kepala website (logo/judul).
  • <nav>: Untuk menu navigasi.
  • <footer>: Untuk bagian kaki website (copyright).
  • <article>: Untuk konten artikel/berita.
Penggunaan:

<header> 
  <h1>Web Sekolah</h1> 
</header>
<nav> 
  <a href="#">Beranda</a> | <a href="#">Profil</a> 
</nav>
<article> 
  <p>Berita hari ini...</p> 
</article>
<footer> 
<p>Copyright 2023</p> 
</footer>
Output:

Secara visual tampak seperti <div>, tetapi secara struktural jauh lebih baik untuk SEO.

Note Penting:

Secara tampilan visual sama saja dengan div, tapi ini membuat kodinganmu lebih profesional dan mudah dibaca mesin pencari.

Penjelasan:

<ul> (Unordered List) digunakan untuk membuat daftar tanpa nomor (biasanya simbol bulat/bullet). Urutannya tidak penting. Tiap isinya pakai tag <li>.

Penggunaan:

<ul>
  <li>Kopi</li>
  <li>Teh</li>
  <li>Susu</li>
</ul>
Output:

Daftar akan ditampilkan dengan simbol (default: bullet point) di depan setiap item.

Note Penting:

Jangan lupa membungkus setiap item dengan <li> (List Item).

Untuk mengubah simbol bullet, gunakan properti CSS list-style-type.

Penjelasan:

<ol> (Ordered List) digunakan untuk membuat daftar bernomor (1, 2, 3...) yang urutannya penting, seperti langkah-langkah tutorial atau juara lomba.

Penggunaan:

<p>Juara Lomba:<p>
<ol>
  <li>Budi</li>
  <li>Ani</li>
  <li>Siti</li>
</ol>
Output:

Juara Lomba akan ditampilkan dengan penomoran otomatis (default: angka).

Note Penting:

Nomor 1, 2, 3 akan muncul otomatis, kamu tidak perlu mengetiknya manual.

Kamu dapat mengubah jenis penomoran menggunakan atribut type (misalnya type="A" untuk huruf kapital).

Penjelasan:

Tag <table> mendefinisikan tabel data. Tag ini berfungsi sebagai wadah untuk semua elemen tabel lainnya.

Penggunaan:

<table border="1">
  <tr>
    <td>Baris 1, Sel 1</td>
    <td>Baris 1, Sel 2</td>
  </tr>
  <tr>
    <td>Baris 2, Sel 1</td>
    <td>Baris 2, Sel 2</td>
  </tr>
</table>
Output:

Tabel yang belum diberi gaya CSS (tampak polos).

Note Penting:

Selalu gunakan tag semantik <thead>, <tbody>, dan <tfoot>.

Gunakan tabel hanya untuk menampilkan data (seperti jadwal pelajaran), jangan untuk mengatur tata letak website.

Penjelasan:

Struktur tabel disusun oleh 3 tag ini:

  • <tr> (Table Row): Membuat baris baru (kiri ke kanan).
  • <th> (Table Header): Judul kolom (otomatis tebal & tengah).
  • <td> (Table Data): Kotak isi data biasa.
Penggunaan:

<table border="1">
  <tr>
    <th>Nama</th> <!-- Judul Kolom -->
    <th>Kelas</th>
  </tr>
  <tr>
    <td>Budi</td> <!-- Isi Data -->
    <td>10 A</td>
  </tr>
</table>
Output:

Sel header (th) secara otomatis akan dicetak tebal.

Note Penting:

Pastikan jumlah kolom (td/th) di setiap baris sama agar tabelnya rapi.

Gunakan atribut colspan atau rowspan pada <td> atau <th> untuk menggabungkan sel.

Bagian 4: Formulir Interaktif

Penjelasan:

Tag <form> digunakan sebagai "amplop" atau wadah untuk membuat formulir (login, register, kontak). Tag ini bertugas mengumpulkan data yang diisi user dan mengirimnya ke server.

Penggunaan:

<form action="/kirim-data" method="POST">
  <!-- Kotak input diletakkan di sini -->
</form>
Output:

Kontainer yang menampung semua elemen input formulir.

Note Penting:
  • action: URL (alamat server) tempat data formulir akan dikirim.
  • method: Metode HTTP yang digunakan (umumnya GET atau POST).

Penjelasan:

<input> adalah kotak isian yang paling serbaguna. Jenisnya bisa diubah-ubah (teks, password, tanggal). <label> adalah judul untuk kotak input tersebut agar user tahu apa yang harus diisi.

Penggunaan:

<label for="nama">Nama:</label>
<input type="text" id="nama" placeholder="Tulis namamu...">

<br><br>

<label for="pass">Password:</label>
<input type="password" id="pass">
Output:

Kotak teks standar dan kotak password (titik-titik) yang siap diisi.

Note Penting:

Hubungkan Label dan Input dengan menyamakan for dan id. Ini membuat label bisa diklik untuk mengaktifkan inputnya.

Penjelasan:

<button> membuat tombol yang bisa diklik. <textarea> membuat kotak teks besar yang ukurannya bisa ditarik, cocok untuk menulis pesan panjang atau komentar.

Penggunaan:

<p>Tulis Pesan:</p>
<textarea rows="4" cols="30">Ketikan pesanmu...</textarea>
<br>
<button type="submit" class="btn btn-primary">Kirim Data</button>
Output:

Tombol kirim (submit) dan kotak besar untuk komentar muncul di halaman.

Note Penting:

Selalu beri atribut type="submit" jika tombol digunakan untuk mengirim formulir.

Selalu tentukan atribut type pada <button>.

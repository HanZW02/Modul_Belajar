Materi Dasar Struktur & Konten

Penjelasan:

Deklarasi ini harus menjadi baris pertama dalam kode HTML kamu. Tujuannya adalah memberi tahu browser bahwa ini adalah dokumen HTML5.

Penggunaan:
Note Penting:

Ini bukan tag HTML, melainkan instruksi untuk browser dan harus diletakkan di baris paling awal. Jadi, tidak perlu untuk membuat </!DOCTYPE html>

Penjelasan:

Tag <html> adalah wadah utama atau akar untuk semua elemen HTML yang lain (kecuali <!DOCTYPE html>).

Penggunaan:
Output:

Tidak memiliki output visual di halaman.

Note Penting:

Sertakan atribut lang (contoh: lang="id") untuk menentukan bahasa dokumen.

Penjelasan:

Tag <head> berisi informasi tentang dokumen (metadata) yang tidak terlihat langsung di halaman, tetapi penting untuk browser, SEO, dan *linking*.

Penggunaan:
Output:

Hanya teks dalam <title> yang terlihat di tab browser.

Note Penting:

Elemen-elemen penting di dalamnya termasuk <title>, <meta>, dan <link> ke CSS.

Penjelasan:

Tag <body> adalah wadah untuk semua konten yang terlihat oleh pengguna di browser.

Penggunaan:
Output:

Semua elemen seperti heading dan paragraf akan terlihat di halaman:

Note Penting:

Setiap dokumen HTML hanya boleh memiliki **satu** tag <body>.

Penjelasan:

Tag <p> mendefinisikan sebuah paragraf. Secara default, browser menambahkan margin (spasi) sebelum dan sesudah paragraf.

Penggunaan:
Output:

Dua blok teks terpisah dengan jarak vertikal di antaranya:

Note Penting:

Jangan gunakan <p> untuk menyusun list atau elemen non-paragraf lainnya.

Penjelasan:

Tag <a> (anchor) membuat hyperlink. Elemen ini mengubah teks atau objek menjadi tautan yang dapat diklik.

Penggunaan:
Output:

Teks yang diklik akan mengarahkan ke URL tujuan:

Note Penting:

Atribut href adalah wajib untuk menentukan alamat tujuan.

Penjelasan:

Tag <img> menyematkan gambar ke dalam halaman. Ini adalah tag *self-closing*.

Penggunaan:
Output:

Gambar yang disematkan akan muncul di halaman:

Note Penting:

Atribut src dan alt adalah wajib. Gunakan width dan height untuk menentukan ukuran.

Penjelasan:

Tag <hr> merepresentasikan garis pemisah tematik (misalnya pergantian subjek) dan ditampilkan sebagai garis horizontal.

Penggunaan:
Output:

Garis horizontal penuh akan muncul di antara konten:

Note Penting:

Ini adalah tag *self-closing* dan digunakan untuk pemisahan konten, bukan untuk dekorasi.

Penjelasan:

Tag <br> menyisipkan jeda baris tunggal (line break). Berguna untuk puisi atau alamat di mana pemisahan baris penting.

Penggunaan:
Output:

Teks akan dilanjutkan di baris baru tanpa membuat paragraf baru:

Note Penting:

Jangan gunakan beberapa tag <br> berturut-turut untuk menambah spasi vertikal. Gunakan CSS Margin atau Padding pada elemen lain.

Penjelasan:

Tag <pre> menampilkan teks dengan format dan spasi yang dipertahankan persis seperti saat diketik dalam kode sumber. Menggunakan font monospace.

Penggunaan:
Output:

Teks akan ditampilkan persis seperti format yang Anda ketik, termasuk spasi dan baris baru:

Note Penting:

Sangat berguna untuk menampilkan blok kode pemrograman atau ASCII Art.

Materi Lanjutan: Atribut, Format Teks, dan Komentar

Penjelasan:

Atribut Global dapat digunakan pada hampir semua elemen HTML untuk memberikan informasi tambahan atau memodifikasi tampilan/perilaku elemen.

Penggunaan:
Note Penting:
  • class: Untuk menata gaya sekelompok elemen (utama untuk CSS/Bootstrap).
  • id: Untuk mengidentifikasi satu elemen unik (penting untuk JavaScript).
  • style: Untuk styling CSS secara inline.
  • title: Menampilkan teks tooltip saat kursor diarahkan ke elemen.

Penjelasan:

Beberapa tag memiliki atribut yang wajib atau sangat penting untuk fungsionalitasnya.

Penggunaan:
Note Penting:
  • **<a>**: target="_blank" membuka tautan di tab baru.
  • **<img>**: alt harus menjelaskan gambar dan sangat penting untuk aksesibilitas.

Penjelasan:

HTML menyediakan tag khusus untuk memformat teks:

  • **Visual (Tidak Semantik):** <b> (Bold), <i> (Italic).
  • **Semantik (Penting):** <strong> (Teks penting), <em> (Penekanan).
  • **Lainnya:** <mark> (Highlight), <del> (Teks dicoret), <ins> (Teks baru), <sub> (Subscript), <sup> (Superscript).
Penggunaan:
Output:

Hasil visual dari semua tag pemformatan teks:

Note Penting:

Sebaiknya selalu gunakan tag **semantik** (<strong> dan <em>) daripada tag visual (<b> dan <i>) karena lebih baik untuk SEO dan aksesibilitas.

Penjelasan:

Komentar digunakan untuk menjelaskan kode, meninggalkan catatan, atau menonaktifkan kode sementara. Komentar **sepenuhnya diabaikan** oleh browser.

Penggunaan:
Output:

Komentar tidak muncul di halaman, hanya terlihat saat melihat sumber halaman (*view page source*).

Note Penting:

Selalu gunakan komentar untuk membuat kode Anda lebih mudah dipahami oleh orang lain (atau diri Anda sendiri di masa depan).

Penjelasan:

Tag <style> digunakan untuk menulis aturan CSS langsung di dalam dokumen HTML. Tag ini ditempatkan di dalam bagian <head>.

Penggunaan:
Output:

Aturan CSS di dalamnya akan mengubah tampilan elemen HTML.

Note Penting:

Penggunaan tag <style> hanya disarankan untuk sedikit aturan. Untuk styling yang lebih besar dan terorganisir, gunakan file eksternal.

Penting: Sintaks dan cara penulisan kode CSS secara lengkap dapat Anda pelajari di **Modul CSS**.
